# RainPath Backend

Backend API for the RainPath case management system built with NestJS.

## Prerequisites

* **Node.js** (v18 or higher)
* **npm** (comes with Node.js)
* **Docker** & **Docker Compose**

## Installation
```bash
npm install
npm run db
npm run start
```

* **GET** `/cases` - Get all cases
* **GET** `/cases/:id` - Get a case by ID
* **POST** `/cases` - Create a new case

## Scripts

* `npm run start` - Start the app in dev mode
